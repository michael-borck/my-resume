# Michael Borck — Resume Site

A simple, accessible resume site built with vanilla HTML, CSS, and JavaScript.

## Features

- Semantic HTML5 — no unnecessary `div`s
- Accessible — skip-link, ARIA-labelled sections, keyboard-friendly
- Responsive — mobile-first layout with print styles
- Zero dependencies — no frameworks, no build step

## Running Locally

Open `index.html` in a browser, or use any static server:

```bash
npx serve .
```

## Deployment

Hosted on GitHub Pages from the `main` branch root.
